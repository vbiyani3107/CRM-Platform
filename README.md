# Bespoke Luxury Platform Documentation

Welcome to the foundation of the Luxury Operating System.

## Documentation Structure

This repository is organized to enforce a strict, process-driven architecture. 

- **[VISION.md](./VISION.md)**: The high-level goals and philosophy of the platform.
- **[docs/product/](./docs/product/)**: Product requirements and atomic work packages (`WORK-PACKAGES.md`).
- **[docs/architecture/](./docs/architecture/)**: Technical specifications, API contracts, data models, and UI design systems.
- **[docs/process/](./docs/process/)**: SDLC guidelines, testing, deployment, and security models.
- **[docs/agents/](./docs/agents/)**: Master prompt library and guardrails for autonomous AI agent development.

Before initiating any development, ensure you read the `AGENT-ANCHOR.md` and the appropriate Work Package specifications.
