# Design System (The Luxury Shell)

This document mandates the visual constraints for the frontend.

## Color Palette
- **Onyx**: `#0f0f0f` (Backgrounds)
- **Champagne**: `#F7E7CE` (Primary Actions)
- **Muted Gold**: `#D4AF37` (Accents)
- **Alabaster**: `#FAFAFA` (Text)

## Typography
- **Headings**: Serif font (e.g., Playfair Display).
- **Body**: Clean Sans-Serif (e.g., Inter or Helvetica Neue).

## Interaction Rules
- **No Clutter**: Generous whitespace.
- **Micro-animations**: Smooth, subtle fades (300ms ease-in-out).
- **Discretion Mode**: Blurring of sensitive names/numbers until explicit hover.
